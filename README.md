# Product Explorer

A product browsing application built with Next.js, TypeScript, and Tailwind CSS.

## Setup Instructions

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open http://localhost:3000 in your browser

## Features

- Product listing with search and category filters
- Product detail pages with ratings and descriptions
- Favorites system with localStorage persistence
- Dark mode support
- Responsive design
